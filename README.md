# use_mdanalysis
This analysis set runs: 
eccentricity, eccentric occupancy, free popc interactions (**IF resname = POPX ONLY), protein-protein interactions (chain and residue), protein-membrane interactions by membrane component and protein residue, distance of protein-membrane midplane

Entire analysis set can be run using the run_analysis.py script

If there is additional analysis you would like added, submit an issue labeled "New Function"

If there is an issue you encounter in any scripts, submit an issue labeled "Error"

If you uncover a problem with script logic/calculations/etc, submit an issue labeled "LOGIC"
